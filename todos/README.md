# HIU Platform Frontend — Roadmap tổng quan

## Mục tiêu
Xây dựng giao diện platform nghe nhạc, chia sẻ ảnh multi-user (kiểu SoundCloud + Instagram).

## Stack
- Framework: Next.js 16 (App Router), React 19, TypeScript
- State: Redux Toolkit + RTK Query
- Styling: Tailwind CSS v3 + React Bits
- Animation: Framer Motion, GSAP
- Icons: Lucide React
